# Predictive Maintenance for Conveyor Belts

ML-powered system that prevents conveyor belt dislodgements and enhances production efficiency using Raspberry Pi and real-time monitoring.

## 🚀 Features

- **Real-time monitoring** of conveyor belt condition using multiple sensors
- **Machine Learning model** to predict belt failures before they occur
- **Raspberry Pi integration** for edge computing
- **Visual and audio alerts** for different severity levels
- **Data logging** for continuous improvement
- **Sensor fusion** combining camera, vibration, temperature, and current data

## 🛠️ Hardware Requirements

- Raspberry Pi 4 (or 3B+)
- Raspberry Pi Camera Module
- DHT11/DHT22 Temperature & Humidity Sensor
- Vibration Sensor (SW-420 or similar)
- Current Sensor (ACS712 or similar)
- LEDs (Green, Yellow, Red)
- Buzzer
- Jumper wires and breadboard
